# Parallel Image Convolution (C/OpenMP)

## Project Overview
Image processing application using discrete convolution with support for custom kernels and performance optimization.

## Features
- Discrete convolution for image filtering
- User-defined convolution kernel
- BMP image format support
- Parallel processing using OpenMP
- SIMD optimization support

## Performance Analysis
- Benchmarking across O0–O3 compiler optimizations
- Multi-core scalability testing
- Execution time measurement

## Output
- Processed images
- Performance comparison results
